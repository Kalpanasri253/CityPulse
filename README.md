# 🏙️ CityPulse

**Infrastructure Health & Anomaly Analysis - Hackathon Project**




## 🚀 Quick Start (30 Seconds)

```bash
pip install flask
python app.py
```

**Open browser:** http://localhost:5000

---

## 📱 Live Dashboard

```
Bridge-01: 🔴 CRITICAL (Score: 10)
Road-01:   🟡 WARNING (Score: 65) 
Pipe-01:   🟢 HEALTHY (Score: 92)

[Update Sensors] 🔄 Click to see changes
```

---

## 🎯 Features

✅ **Real-time monitoring** - Updates every 10 seconds  
✅ **4 Health states** - Green/Yellow/Red scoring
✅ **Anomaly detection** - Vibration spikes, pressure drops  
✅ **Mobile responsive** - Works on phone/tablet
✅ **Production ready** - MQTT sensors compatible

---

## 🛠️ How to Run

```bash
1. pip install -r requirements.txt
2. python app.py
3. Open http://localhost:5000
4. Click "Update Sensors" - Watch health change!
```
